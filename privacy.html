Приложение social_tracker — личный инструмент для сбора статистики просмотров Какие данные используются: - публичные видео и метрики (просмотры) из подключённых аккаунтов через официальные API платформ (TikTok, YouTube, Instagram, VK) - токены доступа хранятся локально в файле .env на компьютере владельца Как используются данные: - только для личного учёта статистики за текущий месяц - данные не продаются, не передаются третьим лицам и не публикуются Хранение и безопасность: - доступ к приложению имеет только владелец аккаунта - токены и ключи не передаются на сторонние серверы Отзыв доступа: - пользователь может отозвать доступ к TikTok и другим платформам в настройках аккаунта на соответствующем сервисе Контакт: augustmrvz (GitHub)
